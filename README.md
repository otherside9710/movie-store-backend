##Docker:
Locate yourself on Root folder in the Project:

Build App: 
``` 
 $ docker-compose build
```

Run App: 
``` 
$ docker-compose up -d
```

Stop App: 
``` 
$ docker-compose down
```

Logs App: 
``` 
$ docker-compose logs -f
```

Show the app running in URL: http://localhost:3000
